App Overview:
 - A simple web application where user enters and finds a drum machine kind of beatpads where users can load sounds and play it.


User Stories:-
 - As a user i want to be able to play the drumpads and should hear the sounds.
 - As a user i want to be able to load the drumpads with my own custom sounds.
 - As a user i want to have multiple functionality such as the click to sound or press to sound. 


In-Scope:-
  - Initial Sound Starter pack.
  - Keyboard mapped sounding beatpad.
  - Load/unload the sounds to each beatpad.
  - Toggle functionalities such as click to sound complete audio file or press to sound file.
  - Volume control functionality.
  - Able to save the pad configuration/ sounds and share it.


Out-Scope:
  - Login functionality.
  - Sequencing.


Technical Constraints:
 - Low latency.
 - Fast responsive, Dynamic ui.
